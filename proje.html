Topographical Feature Extraction Using Machine Learning Techniques from Sentinel-2A Imagery (research project)

Implemented an algorithm using various machine learning techniques like Random forest, SVM, Naive Bayes, Decision tree(CART) satellite images in python. The algorithm can classify the satellite images into 5 classes - Bare land, crop land, water body, road, built up regions,. In the first phase, we classified the satellite image into 5 classes and then in the second phase we classified the crop area into types of crops to get the yield so that we can help the farmers..
